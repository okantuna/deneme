# Ara Dönem Ödevi #1 — Teslim Klasörünüz

Merhaba! Burası sizin teslim klasörünüz. Yazılım/komut bilgisi gerekmez —
yalnızca aşağıdaki adımları izleyip dosyalarınızı koyun.

**Ne yapacaksınız?**
1. Bu klasörün adını ("Ad-Soyad") kendi adınızla değiştirin. Örnek: `Mehmet-Yilmaz`
2. Her klasörün içinde **"buraya-ne-koymalisin.txt"** adlı kısa bir not var. Onu
   okuyun, istenen dosyaları o klasöre koyun.
3. Aşağıdaki bilgileri doldurun.
4. Hepsi tamam olunca klasörü GitHub'a yükleyin (ödev PDF'i, Bölüm 7'deki adımlar).
